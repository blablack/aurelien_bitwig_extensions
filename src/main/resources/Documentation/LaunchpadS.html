Controller documentation

Launchpad S

The mixer button (top right) rotates the different modes available.

Modes

The 4 different modes available are:

  • Piano
  • Diatonic
  • Drums
  • Clip

Piano-Diatonic-Drums

This enables the Launchpad S to play instruments with different keys layout. These 3 modes have a few things in common:

  • The up and down buttons change the octave
  • All the right buttons change the velocity

Piano

The Piano mode provides 4 octaves layed out like a piano keyboard.

Diatonic

The Diatonic mode changes the layout of the keys based on the following 7 diatonic modes:

  • Ionian
  • Dorian
  • Phrygian
  • Lydian
  • Mixolydian
  • Aeolian
  • Locrian

The left and right buttons changes the diatonic mode. The session button rotates the base root.

Drums

The Drum mode provides a layout compatible with the Drum Machine container in Bitwig.

Clips

This mode controls the clip launcher in Bitwig.

The up, down, left and right buttons help to navigate through the the clips.

Each pads can launch or relaunch the corresponding clip.

The right buttons stop the corresponding clip.
