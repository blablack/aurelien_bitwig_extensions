Controller documentation

Launch Control

To switch between templates, press the Factory or User button (top right) and the bottom row of buttons becomes green (Factory) or red (User). Select one of the templates between 1 and 8.

Factory Templates

Only the Factory template 1-3 are supported:

  • Transport Mode
  • Tracks Mode
  • Device Mode

User Templates

The User templates emmits midi events that can be used to midi-learn the knobs in Bitwig

Templates

Factory template 1: Transport Mode

The first button toggles playing.
The second button stops the transport if it is playing, and brings it back to the beginning on the second press.
The third button toggles the recording state.
The fourth button toggles the metronome.

Factory template 2: Tracks Mode

The button 1 and button 5 triggers the recording state for the first and second tracks selected.
The button 2 and button 6 triggers the solo stage for the first and second tracks selected.
The button 3 and button 7 mutes for the first and second tracks selected.

The top knob 1 and top knob 5 controls the volume for the first and second tracks selected.
The bottom knob 1 and bottom knob 5 controls the pan for the first and second tracks selected.

You can change the seleted tracks with the up and down buttons in Arrange layout. The left and right buttons are used in the Mix layout.

Factory template 3: Device Mode

All the knobs are used to controls the Remote Controls of the devices. The knobs 1-4 for the first device, knobs 5-8 for the second.

The button 1 and button 5 turns on/off for the first and second devices selected.
The button 2 and button 6 shows the Remote Controls for the first and second devices selected.
The button 3-4 and button 7-8 are used to navigate through the Remote Control pages for the first and second devices selected.

You can change the seleted tracks with the up and down buttons.
The left and right buttons are used to navigate through the devices.
